Welcome to qlued

Qlued is an open source project to provide access to various quantum hardware simulators. It enables simple access to simulators of various machines that emulate for example Rydberg atoms or hopping fermions.

Using the API

What is needed for v1 of the API

To access the simulator, you will need to use the credentials from this website: username='{{user.username}}' token='YOUR-PASSWORD'. Be aware that v1 does not work with third-party accounts.

What is needed for v2 of the API

In the second version, the access is token based. You will need to use the credentials: username='{{user.username}}' token='{{token_key}}'

Working with the API directly

We have several ways to test the API and find its documentation

  • The documentation of qlued also contains an extensive section on the API.
  • We have an OpenAPI documentation of v1, which is a legacy version
  • We have an OpenAPI documentation of v2, which is the current production version.
